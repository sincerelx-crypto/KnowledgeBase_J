⏺ Per installare Quartz e trasformare il tuo vault di Obsidian in una wiki
  pubblica (che potrai poi dare in pasto a NotebookLM come fonte URL), dobbiamo
  procedere per step.

  Quartz non è un semplice plugin di Obsidian, ma un generatore di siti statici
  che "legge" i tuoi file Markdown e li trasforma in un sito web veloce e
  interconnesso.

  🛠️ Requisiti Preliminari

  Per far funzionare Quartz, hai bisogno di Node.js installato sul tuo computer.


  Controlliamo se l'hai già:
  Posso verificare per te se Node.js è installato. Vuoi che ci provi? (Oppure
  puoi digitare node -v nel tuo terminale).

  ---
  🚀 Guida Passo-Passo all'Installazione

  Se vuoi che ti guidi nell'esecuzione dei comandi, ecco cosa faremo:

  1. Installazione di Quartz

  Useremo il comando ufficiale per creare una nuova istanza di Quartz:
  npx lucide-quartz@latest init
  Questo comando ti chiederà dove vuoi installare Quartz e quale "tipo" di 
  configurazione preferisci (di solito quella "Synced" è la migliore per 
  Obsidian).

  2. Collegamento al Vault di Obsidian

  Una volta installato, dovrai configurare Quartz per puntare alla cartella del
  tuo vault.
  - Dovrai spostare i tuoi file .md nella cartella content di Quartz, oppure
  configurare un link simbolico.

  3. Pubblicazione (Il ponte verso NotebookLM)

  NotebookLM legge meglio i contenuti se sono accessibili via URL o come file.
  Per renderlo una "Wiki" collegabile:
  1. GitHub: Carica il tuo progetto Quartz su un repository GitHub.
  2. GitHub Pages: Attiva GitHub Pages nelle impostazioni del repository.
  3. URL Pubblico: Otterrai un link del tipo
  https://tuo-nome.github.io/tuo-vault/.

  4. Collegamento a NotebookLM

  Una volta che la tua wiki è online:
  1. Apri NotebookLM (https://notebooklm.google.com/).
  2. Crea un nuovo Notebook.
  3. Come "Sorgente" (Source), scegli Website/URL.
  4. Incolla l'URL della tua wiki Quartz.
  5. NotebookLM indicizzerà tutti i tuoi appunti collegati.

  ---
  💡 Come vuoi procedere?

  1. "Fammi vedere se ho Node.js e partiamo": Posso controllare l'ambiente e

  completa dei comandi terminale.
  3. "Aiutami a capire come organizzare il vault prima di iniziare": Se hai
  molte cartelle o plugin complessi, possiamo pianificare la struttura.

  Cosa preferisci?
